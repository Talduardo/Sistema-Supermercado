# 🛒 Sistema de Análise de Supermercado

Este projeto foi desenvolvido em **Python** com o objetivo de analisar como os clientes avaliam um supermercado.  
A aplicação recolhe informações sobre tempo de espera, opiniões sobre políticas internas e sugestões de melhorias, servindo como uma ferramenta de apoio para gestão e tomada de decisão.

---

## 🎯 Objetivos:

- Avaliar a percepção dos clientes em relação ao supermercado.  
- Recolher dados sobre o **tempo perdido nas filas de atendimento**.  
- Registrar a **opinião dos clientes** sobre a política do supermercado.  
- Disponibilizar um espaço para **sugestões de melhorias**.  

---
